What is WebAssembly?

  1. Low-level code for the Web
  2. Binary format
  3. Embedded in JS (WebAssembly object)
  4. A compilation target for C/C++ and Rust

Where does it run?

What can it do?

  1. High performance games (Epic Unreal, Unity)
  2. Audio and Video codecs / 3D model compression
  3. Google Earth
  4. Speech synthesis / recognition
  5. Media applications (Photoshop, After Effects, Maya, Blender, etc.)
  6. Bring existing C/C++ apps to the Web (Linux Stack, Emacs, Vim)
  7. Libraries for cryptography, computer vision, barcode reading
  8. Applications requiring reliable performance, blocking APIs, precise memory control
